1. 项目概述
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

VulkanCalc 2.0 — 基于 Vulkan + GLFW + ImGui 的卡西欧 fx-991CNX 风格科学计算器。
仓库: https://github.com/DedalusArtin/vulkan-calc

技术栈: Vulkan 1.3 / GLFW 3.x / GLM / ImGui / C++20 / CMake
   平台: Linux (WSL) / Windows (VS2022 + vcpkg)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. 已实现的功能 (共 50+ 项)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【核心计算器】
  ✅ 卡西欧 fx-991CNX 风格按钮布局 (7行×5列)
  ✅ 彩色按钮: 数字(灰)、运算符(橙)、函数(蓝)、DEL/AC(红)、=(橙大)
  ✅ LCD 显示屏 (深色背景+亮绿文字)
  ✅ 手动输入 (InputText)+光标控制 (左右方向键)
  ✅ 自动函数匹配 (输入 sin → 弹出建议)
  ✅ 键盘映射: Enter/=/Backspace/ESC/方向键/字母键
  ✅ 角度模式 DEG/RAD/GRAD
  ✅ 进制显示 DEC
  ✅ π常量 (高精度) + e常数
  ✅ 历史记录 (最近20条)
  ✅ Ans 功能
  ✅ 物理常数面板: π, e, c, h, G, NA
  ✅ rand / n! / %

【高级功能 — Chrome标签页模式】
  ✅ 微积分: f(x)求值、定积分(自适应Simpson)、导数
  ✅ 定积分面积填充 (纵向竖线，每3px)
  ✅ 傅里叶级数 (方波/锯齿波/三角波, N项可调)
  ✅ 复变函数: 柯西积分、留数、伽马函数
  ✅ 域着色 (色相=辐角, 亮度=模长)
  ✅ 概率统计: 排列P(n,k)、组合C(n,k)、二项分布、正态分布、泊松
  ✅ 描述性统计 (平均值/中位数/方差/标准差)
  ✅ 统计柱状图 (可调bins、累积频率曲线)
  ✅ 3D曲面图 (等距投影、可旋转缩放)
  ✅ 旋转体体积计算
  ✅ 曲面积分 (蒙特卡洛)

【图像系统】
  ✅ 网格: niceStep算法 (1/2/5/10基数)
  ✅ 零轴加粗 (2.5px + 高亮色)
  ✅ 图像点击 → 全屏大图查看器
  ✅ 拖拽平移 (ImGui::IsMouseDragging)
  ✅ 以鼠标为中心缩放
  ✅ 坐标轴标签 + 图例

【扩展系统】
  ✅ 扩展API架构 (C函数指针)
  ✅ 货币换算 (15种货币 + 实时爬虫 + 缓存)
  ✅ 汇率走势图 (30天模拟)
  ✅ 温度转换 / BMI计算 / 货币换算三个内置扩展
  ✅ open.er-api.com 联网获取实时汇率
  ✅ 网络失败时自动加载缓存

【UI】
  ✅ 中/英/日三语 (自动检测 Windows 语言)
  ✅ 字体: 合并 Default + Greek + ChineseFull + 数学符号 + € + █
  ✅ 标准/高级模式切换 (小米计算器风格)
  ✅ 开机动画 (3D曲面旋转 + 星空粒子 + "开始"按钮呼吸灯)
  ✅ 字体大小调节 S/M/L
  ✅ 窗口自适应缩放

【平台】
  ✅ Linux (WSL) 编译通过
  ✅ Windows (VS2022 + vcpkg) 编译脚本 build-windows.bat
  ✅ CMakeLists.txt 双平台支持
  ✅ resources.rc Windows资源文件
  ✅ installer.iss Inno Setup 安装脚本

3. 文件结构
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

src/
├── main.cpp              # 主入口: 窗口/事件循环/渲染/字体加载/回调
├── state.hpp             # 全局状态: CalcState 结构体
├── locale.hpp            # 多语言: T()函数 + detectLang() + execShell()
├── calc_logic.hpp/cpp    # 计算逻辑: 按钮处理/键盘/表达式/货币/扩展
├── ui.hpp/cpp            # UI渲染: LCD/按钮网格/高级标签页/大图/3D/开机动画
├── math_engine.hpp/cpp   # 数学引擎 (不要修改!) - 积分/微分/复分析/特殊函数
└── vulkan_renderer.hpp/cpp # Vulkan渲染器 (不要修改!)

  build-windows.bat

6. Codex CLI + DeepSeek 不兼容问题
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Codex CLI v0.144.6 只支持 OpenAI 的 Responses API (/v1/responses)。
DeepSeek 只有 /v1/chat/completions，两者不兼容。
解决方案: 使用 Hermes 的 delegate_task 系统（基于 DeepSeek）完成所有代码重写。

7. 注意事项
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 字体加载失败时会 fallback 到默认字体 (但CJK会显示为?)
